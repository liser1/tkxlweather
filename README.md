## 如何使用

<ul>
    <li>克隆项目</li>
    <li>下载依赖与初始化</li>
    <li>运行</li>
    <li>打包</li>
</ul>

## 1.克隆项目

```bash
git clone https://github.com/liser1/tkxlweather.git
```

## 2.下载依赖与初始化

***本项目依赖于Vue.js、需要Node.js以及npm包管理工具***

***当你具备[Node.Js&&npm](https://nodejs.org/)后进入项目根目录执行***

```bash
npm install
## 如果报错请执行下面
npm install --force
```

## 3.运行

```bash
npm run dev
```

## 4.打包

```bash
npm run build
```
